## 複製說明

1. 在 Google Sheets 建立新試算表
2. 建立 4 個 Sheet，分別命名為 `users`、`markets`、`special_dates`、`revenues`
3. 複製上方表格內容（從第一個欄位名稱到最後一筆資料）
4. 在對應 Sheet 的 A1 儲存格貼上
5. 確認欄位正確對齊

## 測試情境

### 登入測試
- `0912345678` → 管理員（王小明）
- `0923456789` → 一般使用者（李小華）
- `0945678901` → 已停用（陳大偉）
- `0999999999` → 不存在

### 租金計算測試
- `2025-01-10`（週五）+ 台北花博 → 400（平日）
- `2025-01-11`（週六）+ 台北花博 → 600（假日）
- `2025-01-01`（週三）+ 台北花博 → 800（特殊日期覆蓋）

### 權限測試
- 李小華（user）登入 → 只能看到自己今天的（測試日需要有資料）
- 王小明（admin）登入 → 可以看全部 + 篩選 + 統計
