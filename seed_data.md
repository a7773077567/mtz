
### 租金規則格式說明

- `1-5:400` = 週一到週五 400 元
- `6-7:600` = 週六日 600 元
- `1-4:350,5:400` = 週一到週四 350 元，週五 400 元
- `1-7:300` = 每天都 300 元
- 留空 = 預設 0 元

**星期對照**：1=週一, 2=週二, 3=週三, 4=週四, 5=週五, 6=週六, 7=週日

## 複製說明

1. 在 Google Sheets 建立新試算表
2. 建立 3 個 Sheet，分別命名為 `users`、`markets`、`revenues`
3. 複製上方表格內容（從第一個欄位名稱到最後一筆資料）
4. 在對應 Sheet 的 A1 儲存格貼上
5. 確認欄位正確對齊
6. **重要**：`手機` 和 `submitted_by_phone` 欄位需設為「純文字」格式，避免手機號碼前導 0 被移除

## 測試情境

### 登入測試
- `0912345678` → 管理員（王小明）
- `0923456789` → 一般使用者（李小華）
- `0945678901` → 已停用（陳大偉）
- `0999999999` → 不存在

### 權限測試
- 李小華（user）登入 → 只能看到自己今天的
- 王小明（admin）登入 → 可以看全部 + 篩選 + 統計
